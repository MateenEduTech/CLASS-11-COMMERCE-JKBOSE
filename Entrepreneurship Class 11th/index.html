About the App

This interactive Progressive Web App covers the complete JKBOSE Class 11 Entrepreneurship syllabus. It is designed in alignment with NEP 2020, NCF, PARAKH, and NCERT guidelines.

The app runs 100% offline and can be installed as a mobile app on Android devices.
